Get started

Using npx

  • 1. Install and create your library immediately:

    npx javali my-lib
  • 2. Enjoy

Using global cli

  • 1. Install Javali globally:

    yarn global add javali or npm install -g javali

  • 2. Create your library:

    javali my-lib
  • 3. Enjoy
